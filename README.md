vis-1
=====

slides for denver/boulder d3/vis meetup

To run
======

Install [node](http://nodejs.org).

Clone this repository.

At a command line:
```
npm install -g yo bower grunt-cli
npm install
bower install
grunt server
```

This should open a new browser tab with the slides.
